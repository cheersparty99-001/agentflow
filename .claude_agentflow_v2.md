### Key Implementation Details (READ CAREFULLY)

1. **Demo Mode**: `config.py` has `DEMO_MODE = os.getenv("DEMO_MODE", "true").lower() == "true"`. When DEMO_MODE is True, supabase errors are silently caught and default values returned. `main.py` initializes `app.state.demo_logs = []` on startup — all demo logs are stored there as a list of dicts.

2. **Auth**: Session-based via `itsdangerous.URLSafeTimedSerializer`. `get_current_user(request)` reads from cookies. Demo login: email=`demo@agentflow.my`, session includes `user_id`, `account_id` (UUID `00000000-0000-0000-0000-000000000001`), `role`, `email`.

3. **Safe Supabase helpers** in `services/supabase_client.py`: `safe_single`, `safe_multi`, `safe_count`, `safe_insert`, `safe_update`, `safe_delete` — all catch exceptions gracefully in demo mode.

4. **Jinja2**: Templates use `Environment(loader=FileSystemLoader("templates"))` instantiated in each router file. Variables passed: `agency_name`, `demo_mode`, `current_path`.

5. **Demo policies**: Stored only in schema.sql seed data. The policies router fetches from Supabase; in demo mode when Supabase is unreachable, it returns empty arrays.

**static/i18n.js:**
```javascript
(function() {
  const lang = localStorage.getItem('agentflow_lang') || 'en';
  document.documentElement.lang = lang;

  window.setLang = function(code) {
    localStorage.setItem('agentflow_lang', code);
    location.reload();
  };

  async function applyTranslations() {
    const lang = localStorage.getItem('agentflow_lang') || 'en';
    try {
      const res = await fetch('/locales/' + lang + '.json');
      const dict = await res.json();
      document.querySelectorAll('[data-i18n]').forEach(el => {
        const key = el.getAttribute('data-i18n');
        if (dict[key]) {
          if (el.tagName === 'INPUT' && el.getAttribute('placeholder')) {
            el.placeholder = dict[key];
          } else {
            el.textContent = dict[key];
          }
        }
      });
    } catch(e) {}
  }
  document.addEventListener('DOMContentLoaded', applyTranslations);
})();
```
